C++ htslib/bwa-mem interface and command line tools for interrogating BAM and SAM files.

License: GNU GPLv3

Description

SeqLib is a C++ package for querying BAM and SAM files, performing BWA-MEM operations in memory, and performing sequence assembly with Fermi from Heng Li. At its core, SeqLib is a C++ interface to the C htslib, bwamem and fermikit projects, which provides the code for core operations. (https://github.com/samtools/htslib)

SeqLib is built to be extendable. See VariantBamfor examples of how to take advantage of C++ class extensions to build off of the SeqLib base functionality.

SeqLib is available as a stand-alone executable, which wrap a number of tools. However, certain functionality provided in an optimal form by SamTools or BamTools are intentially left out of SeqLib. In cases where improvements or alternative approaches are useful, the functionally is partially duplicated (e.g. count).

Installation

 git clone https://github.com/jwalabroad/SeqLib.git
 cd SeqLib
 ./configure
 make

I have successfully compiled with GCC-4.9 on Linux.
