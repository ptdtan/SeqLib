1 /* genomeRangeTree - This module is a way of keeping track of
2  * non-overlapping ranges (half-open intervals) across a whole
3  * genome (multiple chromosomes or scaffolds).
4  * It is a hash table container mapping chrom to rangeTree.
5  * Most of the work is performed by rangeTree, this container
6  * enables local memory and stack to be shared by many rangeTrees
7  * so it should be able to handle genomes with a very large
8  * number of scaffolds. See rangeTree for more information. */
