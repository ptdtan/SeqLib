Detailed Description

Class to store and interact with an HTSLib bam1_t read.

HTSLibrary reads are stored in the bam1_t struct. Memory allocation is taken care of by bam1_t init, and deallocation by destroy_bam1. This class is a C++ interface that automatically takes care of memory management for these C allocs/deallocs. The only member of BamRead is a bam1_t object. Alloc/dealloc is taken care of by the constructor and destructor.
