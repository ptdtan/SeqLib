1 // dsPrint - Dynamic string Stack based Printing.
2 //
3 // This module relies upon a dyString based stack of buffers which accumulate output
4 // for later printing. As a stack, only the top buffer can be filled and flushed out
5 // (dsPrintFlush). When flushing, the top buffer's content is appended to the next
6 // lower buffer, unless explicitly requested otherwise (dsPrintDirectly).
7 // If the stack is empty, dsPrintf will be equivalent to printf.
8 //
9 // Output goes to STDOUT unless a single alternative out file is registered.
