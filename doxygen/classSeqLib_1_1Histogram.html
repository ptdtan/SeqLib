Detailed Description

Class to store histogram of numeric values.

The bins of the Histogram are not uniformly spaced, and their ranges determined by partitioning the spans it tablulates into uniform quantiles when initialized by Histogram::initialSpans(). As elements are added and removed this initial bin definition remains constant.
