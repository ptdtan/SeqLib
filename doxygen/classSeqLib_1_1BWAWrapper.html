Detailed Description

Calls BWA-MEM on sequence queries and returns aligned reads, all in memory.

Note
Calls core functions provided by Heng Li in BWA-MEM. https://github.com/lh3/bwa

Constructor & Destructor Documentation

BWAWrapper ( )
inline

Create an empty BWA MEM interface.

Note
Will initalize a BWA-MEM memopt structure with the BWA-MEM defaults found in mem_opt_init. Memory allocation and deallocation is automatically handled in constructor / destructor.
