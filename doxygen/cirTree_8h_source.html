1 /* cirTree chromosome id r tree. Part of a system to index chromosome ranges - things of
2  * form chrN:start-end. Generally you'll be using the crTree module - which
3  * makes use of this module and the bPlusTree module - rather than this module directly.
4  * This module works with chromosomes mapped to small integers rather than chromosomes
5  * as strings, saving space and speeding things up in the process, but requiring the
6  * separate bPlusTree to map the names to IDs.
7  * This module implements a one dimensional R-tree index treating the chromosome ID
8  * as the most significant part of a two-part key, and the base position as the least
9  * significant part of the key. */
