1 /* bigWig - interface to binary file with wiggle-style values (that is a bunch of
2  * small non-overlapping regions each associated with a floating point value.
3  *
4  * There are several ways to use this module. To create a new bigWig file use the
5  * bigWigCreate function, which takes as input an ascii file in fixedStep, variableStep
6  * or bedGraph format.
7  *
8  * To get a section of of a bigWig for display on a browser, use the bigWigSummaryArray
9  * function, which will fill in an array of doubles with on value for each pixel that
10  * you want to display.
11  *
12  * To read all the data out of a bigWig get the chromosome info with bbiChromList
13  * and then fetch all of it for each chromosome using bigWigIntervalQuery.
14  *
15  * See also the module bbiFile that has a description of they structure of
16  * a bigWig file, and lower level routines used to implement this interface.
17  */
