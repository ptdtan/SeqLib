Detailed Description

Store information pertaining to a given read group *.

This class will collect statistics on number of: read, supplementary reads, unmapped reads, qcfail reads, duplicate reads. It will also create Histogram objects to store counts of: mapq, nm, isize, clip, mean phred score, length
