1 /* basicBed.h contains the basic interface to Browser Extensible Data (bed) files and tables.
2  * The idea behind bed is that the first three fields are defined and required.

4  * In addition after any number of defined fields there can be custom fields that
5  * are not defined in the bed spec.
6  *
7  * There's additional bed-related code in src/hg/inc/bed.h. This module contains the
