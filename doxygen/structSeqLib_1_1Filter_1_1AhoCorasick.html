Detailed Description

Tool for using the Aho-Corasick method for substring queries of using large dictionaries.

Note
Trie construction / searching implemented by https://github.com/blockchaindev/aho_corasick

inline

Add a motif to the trie.

Note
Trie construction is lazy. Won't build trie until first query. Therefore first query is slow, the rest are O(n) where (n) is length of query string.
