Constructor & Destructor Documentation

~BamReader ( )
inline

Destroy a BamReader and close all connections to the BAM.

Calling the destructor will take care of all of the C-style dealloc calls required within HTSlib to close a BAM or SAM file.

bool SetCramReference ( const std::string &  ref)

Explicitly set a reference genome to be used to decode CRAM file.

If no reference is specified, will automatically load from file pointed to in CRAM header using the tags.

Note
This function is useful if the reference path pointed to by the UR field of is not on your system, and you would like to explicitly provide one.
